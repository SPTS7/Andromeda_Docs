# Andromeda Server

## Hardware

!!! info "System Specifications"
    - **Model**: Dell Precision 3630 Tower
    - **CPU**: Intel Core i7-8700K @ 3.70GHz (6c/12t)
    - **RAM**: 32 GB DDR4
    - **OS**: Debian 12 (Bookworm) / Proxmox VE 8.4.19
    - **Kernel**: Linux 6.8.12-18-pve
    - **SSD**: 512 GB NVMe (Patriot M.2 P320) — System pool
    - **HDD**: 1 TB (WDC WD10EARS) — Media storage
